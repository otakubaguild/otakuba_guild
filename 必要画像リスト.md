# おたく場ギルド系テンプレート｜必要画像リスト

このアプリはコンセプト（RPG／宇宙・SF／魔法学校）を切り替えて使えます。
各コンセプトで下記の画像を用意し、指定のフォルダに置くと、切り替えた時に表示されます。
画像が無い間は、敵は👾（絵文字）で代替表示されます。

## 画像の共通仕様（推奨）

- **敵画像**：背景が透明なPNG。正方形〜縦長（目安 512×512〜768×1024px）。1枚あたり2MB以下推奨。
- **背景画像**：横長のPNGまたはJPG（目安 1200×800px前後）。1枚2〜3MB以下推奨。
- **マスター画像**：正方形PNG（目安 400×400px）。店の“案内役”の顔。
- ファイル名は**半角英数字**のまま（変えると表示されません）。
- 1ファイル10MB以下。フォルダごとGitHubに置けばOK。

## 音（BGM）について

BGMは3コンセプト共通で、今あるもの（slime / goblin / orc / cave / ruins / maou / daimaou など）を流用します。
コンセプト専用のBGMにしたい場合は、mp3を差し替えるか、討伐設定の「BGM URL」欄に音源URLを貼ってください。

## メモ

- 敵画像は「1体につき1枚」。同じ敵が複数いる場合（例：ゴブリン1/2・2/2）は同じ画像を使い回します。
- 画像を用意したら、管理画面の「🎭 コンセプト」タブで切り替えると反映されます。
- まず1コンセプトだけ画像を揃えて試すのがおすすめです。
